# Analyze-Email-Headers-for-IP-Reputation
Analyze Email Headers for IP Reputation and Spoofing Detection (Gmail + n8n)
single-page application (SPA) infographic that visualizes the n8n security workflow guide.

**File Summary:**
* **`n8n_security_infographic.html`**: A self-contained HTML file containing the complete infographic. It utilizes Tailwind CSS for layout and styling, Chart.js and Plotly.js for data visualization, and custom JavaScript for logic handling (such as label wrapping). It strictly adheres to the "No SVG" and "No Mermaid" constraints, using CSS and Unicode for diagrams.

### How to View
Download the file and open it in any modern web browser.
